..::: Minimal Respaldo Autonomo Basico :::..

 Este programa es para los que tienen prisa
 en salvar solo lo necesario en su equipo.

 MRAB no crea avisos o mensajes al  momento
 de copiar los archivos, esto permite poder
 usarlo de forma silenciosa (en caso que se
 este usando de forma remota).
